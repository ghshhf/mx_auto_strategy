# mx_auto_strategy 回测报告（修正版：真实持仓标的，非指数代理）

## 摘要

本报告是对此前"指数代理回测"的修正版（v3，每周再平衡校正）。用户的两条关键纠正已落地：(1) 系统持有的是**具体股票/可转债**，不是指数；(2) 弱市（防御）模式下进攻仓不是一直满仓高 Beta 股票，而是按代码切到**可转债（债底保护）**，近似"只买防御股 + 大量现金、没有真进攻"。修正后用真实标的历史（2016-07 至 2026-07，akshare Sina 源）重跑，起步 1 万元、周频再平衡，并新增 10 年维度。结论：即便用保守防御篮，5 年约 2.49 万元（+149%，年化 20%，回撤 -25.6%）、10 年约 9.77 万元（+877%，年化 25.6%），大幅跑赢买入持有沪深300（10 年 +46%）；弱市进攻切转债使回撤从错误版的 -31% 降到 -23%~-26%，且正是避开 2022 年宁德/凯莱英 -78% 个股崩盘的结果。本回测仍是**配置层代理**（进攻仓为代表性真实标的篮，非逐笔重放），详见局限。

## 相对上一版（指数代理）的关键修正

- 防御仓不再用沪深300 指数，而用真实低 Beta 蓝筹：乐观版取长江电力、招商银行、中国神华、宁沪高速、贵州茅台、中国移动 6 只（配置中标注"防御/避险/高股息/弱市不跌"的核心）；保守版扩到 16 只池内代表股（补民生银行、平安银行、中信证券、万科A、隆基绿能、中国建筑、海螺水泥、格力、美的、伊利，含分化弱势票以稀释"只挑赢家"偏差）。
- 进攻仓分两段：弱市按 `auto_trader.py` 真实逻辑切到**可转债**（南银转债 113050、兴业转债 113052，债底保护、非股票）；平衡/强势用真实高 Beta 代表篮（宁德时代、中际旭创、北方华创、凯莱英）。这正对应你说的"防御模式没有真进攻"。
- regime 信号仍为沪深300 周线相对 20 周均线 ±3%（对齐 `selector.market_regime` 与 config 的 `defensive_benchmark`/`regime_band_pct`），指数只用于"判市信号"，不作为持仓。

## 弱市 + 温度计下的真实仓位（验证你的直觉）

代码 `REGIME_ALLOC`：弱市 def 60 / off 24 / cash 16。但弱市进攻仓已被替换成可转债（低波动），且温度计在偏冷时进一步削进攻转现金。算下来：

- 弱市 + 常温（温度 ≥55）：约 60% 防御股 + 24% 可转债 + 16% 现金。
- 弱市 + 偏冷（温度 30~55）：进攻仓 ×~0.7 → 约 60% 防御股 + ~17% 可转债 + ~23% 现金。
- 弱市 + 冰点/脆弱（温度 <30 且缩量）：进攻倍率 0.24~0.4 → 约 60% 防御股 + 6%~10% 可转债 + 30%~34% 现金。

即深度弱市下组合确实接近"防御股为主 + 三成现金 + 极少可转债"，与你说的"防御模式只有现金很多、只买防御股、没有进攻"高度一致（差异仅在于代码仍保留约 6%~24% 的可转债类进攻，而非绝对清零；若你想在弱市直接把进攻降到 0，那是改 `REGIME_ALLOC` 的 `weak.off` 配置，目前是 24%）。

## 温度计（方案 C）的边际效果（如实说明）

在修正框架上，温度计的增量影响较小且明确是"以收益换平滑"、而非增收：10 年回撤 -23.7%→-23.0%、期末市值 160916→129245（收益降约 20%）；5 年回撤 -23.7%→-23.0%、收益略降；3 年回撤 -10.3%→-9.2%、收益略降。原因是——弱市进攻仓**已经切到可转债**避开了最狠的个股崩盘，温度计是在此之上对进攻时机再做微调，本样本里主要是平滑回撤而非增收。换言之，本回测里"大胜买入持有"主要来自框架本身（防御蓝筹 + 弱市切转债），温度计是叠加的风险调速器，**本质就是让防御更平滑，不是去增强进攻**——这与你的判断一致，不应被夸大为收益来源。

## 重要假设与局限（看数前必读）

- **仍是配置层代理，非逐笔交易重放**。真实 alpha 来自 `weekly_theme.pick_theme()` 每周动态选的 2 只进攻股，该链路无历史数据源可复现；平衡/强势进攻篮用的是"真实高 Beta 代表样本"（宁德/中际旭创/北方华创/凯莱英），其波动/收益不等于系统实际每周选股结果，可能偏高也可能偏低。
- **防御篮构造偏差**：乐观版 6 只是人工挑的强红利赢家（2021-2026 高股息行情中表现极佳），会高估防御收益；保守版 16 只稀释了该偏差，给出下界。真实系统按 3D 评分每周轮动 Top3，5 年里也会踩到弱势票，实际应介于两者之间、更靠近保守版。
- **未计交易成本/印花税/滑点**：按周频再平衡，年化摩擦约 0.5%~1.5%，会进一步侵蚀上表收益。
- **单一样本区间依赖**：仅 1 个 5 年 + 1 个 3 年区间，且都包含 2021-2024 熊市与 2024-2026 红利/科技修复，结论存在样本区间依赖；严格结论需 walk-forward / 样本外验证，切忌过拟合。
- **两融维度不可得**：温度探针的"融资余额"维度在标的历史中无法获取，按方案 C 稳健设计"维度失败即排除并权重重归一化"处理（宽度/量能/趋势按 30/30/25 归一到 100%），不影响安全结论。

## 对实盘的含意

- 框架本身（防御蓝筹为底 + 弱市进攻切转债 + 周频再平衡）在 2021-2026 这个"蓝筹/红利占优、成长大起大落"的区间内表现远好于买入持有宽基，且回撤显著更低——这支撑了你"守正出奇"的主线设计。
- 若想更贴近你描述的"弱市几乎零进攻"，可把 `strategy_config.json` 里 `REGIME_ALLOC.weak.off` 从 24 降到 0~10（同时把 freed 部分加进 cash），回测会进一步降回撤、但也会在弱市转暖时少赚反弹。要我改配置并重跑就一句话。
- 温度计（方案 C）建议保留为"风险调速器/影子观察"，不要期待它显著增收；它的价值在极端脆弱市替你收力，而非创造 alpha。
