核心结论：16倍框架（防御蓝筹底仓 + 弱市进攻切可转债 + 周频再平衡）10 年跑出约 16.1 万（+1509%），而叠加「多指数周线死叉 ≥3 转全防御」后 10 年约 16.8 万（+1577%），同时最大回撤从 -23.7% 压到 -20.9%。即：收益略升、回撤更浅。

逻辑边界：死叉信号只是去风险层，不是收益引擎。触发时把进攻仓清零、转入防御蓝筹篮；已持仓的进攻票仍按原有止盈/止损持有（不强行一键清仓），遵循「周频不全卖」的铁律。未计交易成本/滑点。

16倍来源：16倍 = 你的 OFF4 精选（宁德/中际旭创/北方华创/凯莱英） + DEF16 红利赢家（长江电力/招行/神华/宁沪/茅台/伊利等）按 regime 周频再平衡。新系统是在这一框架上合规叠加死叉层，不是凭空升逻辑。
