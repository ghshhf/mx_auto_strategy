# 用户剧本入口（User Script）

> **这个文件就是"大方向直接给剧本"的入口。**
> 你直接用人话写下周方向，系统每周运行 `auto_trader.py` 时自动读取并执行。
> 写什么，系统演什么。（剧本书写者模式）

---

## 写法规则（给"其他AI"或你自己参考）

直接写自然语言即可，系统解析以下关键词：

| 你写的 | 系统理解 |
|---|---|
| `电力` / `电网` | 进攻端叠加电力方向（长江电力/中国核电/南网储能/国电南瑞） |
| `医疗` / `医药` | 进攻端叠加医药方向（迈瑞/恒瑞/药明/医药ETF） |
| `防御端自己定` / 不写 | 系统从防御白名单自动选Top3，三档调节 |
| `不给方向` / 不写进攻 | 系统走自适应主线（scan_industry_momentum） |
| `剧本` / `大方向` | 本文件即剧本，照此执行 |

**示例写法：**
```
下周主攻电力和医疗，防御端你定，弱势市多留现金。
```
系统会自动：
1. 检测本文件 mode → 设为 user_direction_overlay
2. 进攻端从电力/医药里挑动量最强票
3. 防御端系统自定（银行+电力+红利低波）
4. 弱势市自动降进攻仓位、抬现金
